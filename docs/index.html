IMAF Muxer / Demuxer

Pure TypeScript muxer/demuxer for ISO/IEC 23000-12 (IMAF) with MPEG-7 metadata and minimal ISO-BMFF builders.

API Reference GitHub Repo

What it does

  • Mux AAC/ADTS, MP3, WAV/PCM (SAOC experimental) + tx3g → .imaf (MP4)
  • Demux streams, extract MPEG-7 JSON, convert tx3g ⇄ SRT
  • Author/read IMAF groups, presets, rules (grco/prco/ruco)

Quick start

# Mux
npx tsx scripts/mux-imaf.ts --out out.ima --in a.aac --in b.aac --meta meta/song.json

# Demux
npx tsx scripts/demux-imaf.ts --in out.ima --out-dir extracted/
